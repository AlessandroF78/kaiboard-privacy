Politique de Suppression des Données

kAIboard respecte votre droit à la suppression des données conformément à l'article 17 du RGPD (Droit à l'effacement). Vous pouvez supprimer vos données personnelles à tout moment, soit directement via l'application, soit en nous contactant par e-mail.

Méthode 1 : Suppression dans l'Application (Immédiate)

Le moyen le plus rapide de supprimer vos données est directement via l'application :

  1. Ouvrez kAIboard sur votre appareil Android
  2. Accédez aux Paramètres
  3. Sélectionnez Confidentialité et Sécurité
  4. Choisissez l'une des trois options de suppression

Options de Suppression Disponibles

Option Ce qui est Supprimé Effet
Supprimer les Données Cloud Solde de jetons et historique d'achat stockés dans Firebase Irréversible. Vous perdrez votre solde de jetons.
Supprimer les Données d'Apprentissage Mots appris pour les suggestions personnalisées (toutes les 11 langues) Le texte prédictif repartira de zéro.
Supprimer Toutes les Données Locales Toutes les préférences, paramètres et données locales L'application redémarrera avec les paramètres par défaut.

Délai de Traitement : Immédiat (0-5 secondes)

Méthode 2 : Demande par E-mail

Si vous préférez une assistance ou avez des questions sur la suppression des données, vous pouvez nous contacter par e-mail :

E-mail : support@kaiboard.eu

Objet : "Demande de Suppression de Données RGPD"

Informations à Inclure :

  • ID de l'appareil (si connu) ou description de l'appareil
  • Type de données que vous souhaitez supprimer (cloud, données d'apprentissage ou toutes)
  • Confirmation de votre identité (pour des raisons de sécurité)

Délai de Réponse : Maximum 30 jours (comme l'exige l'article 12 du RGPD)

Suppression Automatique des Données

kAIboard inclut des mécanismes de nettoyage automatique des données pour protéger votre vie privée :

Type de Donnée Période de Conservation Détails
Données Cloud (Firebase) 365 jours d'inactivité Supprimées automatiquement après un an sans utilisation de l'application
Mots Appris Indéfini avec dégradation Les scores de fréquence diminuent avec le temps ; les mots inutilisés sont supprimés
Cache de Traduction Session uniquement Stocké uniquement en RAM ; supprimé à la fermeture de l'application
Texte de Traduction Jamais stocké 0 seconde de conservation ; transit uniquement vers l'API OpenAI

Données Non Supprimables

Pour des raisons légales et techniques, certaines données ne peuvent pas être supprimées immédiatement :

  • Transactions Google Play : Gérées par Google selon leurs politiques (pour les demandes de remboursement, la prévention de la fraude et la conformité fiscale)
  • Journaux Système Anonymes : Conservés jusqu'à 90 jours à des fins de débogage (ne contiennent aucune information personnelle identifiable)

Conséquences de la Suppression des Données

Avis Important : La suppression de certaines données est irréversible. Assurez-vous de comprendre les conséquences avant de continuer :

  • Supprimer les Données Cloud : Vous perdrez votre solde de jetons et ne pourrez PAS le récupérer, même après avoir réinstallé l'application
  • Supprimer les Données d'Apprentissage : Les prédictions de texte personnalisées repartiront de zéro
  • Supprimer Toutes les Données Locales : Toutes les préférences et paramètres seront perdus

Questions Fréquemment Posées

Que se passe-t-il si je supprime les données cloud ?

Vous perdrez votre solde de jetons et votre historique d'achat. Si vous réinstallez l'application, vous repartirez de zéro.

Puis-je supprimer uniquement des données spécifiques ?

Oui. Vous pouvez choisir de supprimer uniquement les données cloud, uniquement les données d'apprentissage ou toutes les données ensemble.

Mes messages traduits sont-ils enregistrés ?

Non. Le texte que vous saisissez pour la traduction n'est jamais enregistré. Il est envoyé à OpenAI uniquement pour le traitement de la traduction et immédiatement supprimé (0 seconde de conservation).

Combien de temps prend la suppression des données ?

La suppression dans l'application est immédiate (0-5 secondes). Les demandes par e-mail sont traitées dans un délai de 30 jours.

Puis-je récupérer les données après la suppression ?

Non. La suppression est irréversible. Assurez-vous de vouloir supprimer vos données avant de continuer.
