2. Service description

kAIboard is a multilingual keyboard app for Android with the following core features:

  • Keyboard layouts for currently 14 languages (subject to expansion)
  • Automatic language detection based on the User’s contacts (phone number prefix)
  • Offline autocorrect using Hunspell dictionaries (local on the device)
  • Offline word prediction using KenLM language models (downloaded on demand)
  • Offline translation using ML Kit language packs (downloaded on demand, approx. 30 MB per language)
  • AI-powered online translation (optional, via GPT-4o-mini through a Cloudflare Worker Relay)

The available features depend on the chosen subscription plan (see section 4). The Provider reserves the right to expand or modify the features at any time.
