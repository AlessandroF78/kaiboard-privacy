Política de Eliminação de Dados

kAIboard respeita o seu direito à eliminação de dados nos termos do Artigo 17 do RGPD (Direito ao Apagamento). Pode eliminar os seus dados pessoais a qualquer momento, diretamente através da aplicação ou contactando-nos por e-mail.

Método 1: Eliminação na Aplicação (Imediata)

A forma mais rápida de eliminar os seus dados é diretamente através da aplicação:

  1. Abra o kAIboard no seu dispositivo Android
  2. Vá a Definições
  3. Selecione Privacidade e Segurança
  4. Escolha uma das três opções de eliminação

Opções de Eliminação Disponíveis

Opção O Que é Eliminado Efeito
Eliminar Dados da Nuvem Saldo de tokens e histórico de compras armazenado no Firebase Irreversível. Perderá o seu saldo de tokens.
Eliminar Dados de Aprendizagem Palavras aprendidas para sugestões personalizadas (todos os 11 idiomas) O texto preditivo começará do zero.
Eliminar Todos os Dados Locais Todas as preferências, definições e dados locais A aplicação reiniciará com as definições padrão.

Tempo de Processamento: Imediato (0-5 segundos)

Método 2: Pedido por E-mail

Se preferir assistência ou tiver questões sobre a eliminação de dados, pode contactar-nos por e-mail:

E-mail: support@kaiboard.eu

Assunto: "Pedido de Eliminação de Dados RGPD"

Informação a Incluir:

  • ID do dispositivo (se conhecido) ou descrição do dispositivo
  • Tipo de dados que deseja eliminar (nuvem, dados de aprendizagem ou todos)
  • Confirmação da sua identidade (por motivos de segurança)

Tempo de Resposta: Máximo 30 dias (conforme exigido pelo Artigo 12 do RGPD)

Eliminação Automática de Dados

O kAIboard inclui mecanismos de limpeza automática de dados para proteger a sua privacidade:

Tipo de Dados Período de Retenção Detalhes
Dados da Nuvem (Firebase) 365 dias de inatividade Eliminados automaticamente após um ano sem uso da aplicação
Palavras Aprendidas Indefinido com degradação As pontuações de frequência diminuem ao longo do tempo; palavras não utilizadas são removidas
Cache de Tradução Apenas sessão Armazenado apenas na RAM; eliminado ao fechar a aplicação
Texto de Tradução Nunca armazenado 0 segundos de retenção; apenas trânsito para a API OpenAI

Dados Não Elimináveis

Por razões legais e técnicas, certos dados não podem ser eliminados imediatamente:

  • Transações do Google Play: Geridas pelo Google de acordo com as suas políticas (para pedidos de reembolso, prevenção de fraude e conformidade fiscal)
  • Registos do Sistema Anónimos: Retidos até 90 dias para fins de depuração (não contêm informação pessoal identificável)

Consequências da Eliminação de Dados

Aviso Importante: A eliminação de certos dados é irreversível. Certifique-se de compreender as consequências antes de prosseguir:

  • Eliminar Dados da Nuvem: Perderá o seu saldo de tokens e NÃO poderá recuperá-lo, mesmo após reinstalar a aplicação
  • Eliminar Dados de Aprendizagem: As previsões de texto personalizadas começarão do zero
  • Eliminar Todos os Dados Locais: Todas as preferências e definições serão perdidas

Perguntas Frequentes

O que acontece se eu eliminar os dados da nuvem?

Perderá o seu saldo de tokens e histórico de compras. Se reinstalar a aplicação, começará do zero.

Posso eliminar apenas dados específicos?

Sim. Pode escolher eliminar apenas os dados da nuvem, apenas os dados de aprendizagem ou todos os dados juntos.

As minhas mensagens traduzidas são guardadas?

Não. O texto que escreve para tradução nunca é guardado. É enviado para a OpenAI apenas para processamento de tradução e imediatamente descartado (0 segundos de retenção).

Quanto tempo demora a eliminação de dados?

A eliminação na aplicação é imediata (0-5 segundos). Os pedidos por e-mail são processados em 30 dias.

Posso recuperar os dados após a eliminação?

Não. A eliminação é irreversível. Certifique-se de que deseja eliminar os seus dados antes de prosseguir.
